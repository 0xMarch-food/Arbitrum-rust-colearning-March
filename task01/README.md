## 1.切换Arbitrum Sepolia网络
> 我是在官方文档 https://docs.arbitrum.io/build-decentralized-apps/quickstart-solidity-remix 里面找到的信息
- 网络名称：`Arbitrum Sepolia`
- 新的 RPC URL：`https://sepolia-rollup.arbitrum.io/rpc`
- 链 ID：`421614`
- 货币符号：`ETH`

切换步骤：在“代币”板块下选择Custom网络，最下方点击自定义网络，将上面的信息输入对应的框框里，即可添加该网络。
